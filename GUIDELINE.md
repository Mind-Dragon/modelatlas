# ModelAtlas Guideline

## Human Vision

The world's most comprehensive open database of AI model providers, their models,
pricing, capabilities, and API endpoints. Updated autonomously. Zero human touch
after initial setup.

## Non-negotiables

- GUIDELINE.md owns project intent.
- ARCHITECTURE.md must derive from this file.
- PLANS.md owns version-to-version waves.
- TODO.md owns only the current executable iteration.
- AUTONOMOUS.md owns swarm rules.
- No secrets in public files. Ever.
- schema.sql is DDL-only. seed.sql is DML-only. No mixing.
- All data must cite its source. No unsourced pricing.
- Every release must pass verify-release.py before merging.
